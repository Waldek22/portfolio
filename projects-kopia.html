Strona Meble Furni jest responsywną stroną stworzoną na kursie front-endu od Kodilli. Jest to pierwsza wykonana przeze mnie strona. Prosta jednostronicowa wizytówka może służyć jako strona informacyjna dla firmy sprzedającej meble. Została stworzona w celu przećwiczenia tworzenia stron opartych na gridzie.

  • Html, CSS
  • Sass
  • Responsive web design
  • Oparta na gridzie
